# 蓝图节点文本参考

> UE 蓝图事件图（EventGraph）的文本序列化格式参考
> 来源：`BP_FirstPersonCharacter` 蓝图资源
> 格式：与 `.uasset` 反序列化后的节点文本输出一致

## 结构说明

文件由多个 `Begin Object ... End Object` 块组成，每个块代表蓝图中的一个节点。

### 节点类型

| 类型 | 说明 |
|------|------|
| `K2Node_CallFunction` | 函数调用节点 |
| `K2Node_EnhancedInputAction` | 增强输入动作节点 |
| `K2Node_Event` | 事件节点 |
| `K2Node_FunctionEntry` | 自定义函数入口 |
| `EdGraphNode_Comment` | 注释框节点 |
| `K2Node_Knot` | 连线转接节点（Reroute） |

### 节点标准字段

1. **Begin Object 行**：`Class`、`Name`、`ExportPath`
2. **FunctionReference**（函数节点）：`MemberName`、`bSelfContext`
3. **InputAction**（输入节点）：输入动作资源路径
4. **NodePosX / NodePosY**：画布坐标
5. **NodeGuid**：唯一标识符
6. **CustomProperties Pin**：引脚定义（PinId/PinName/PinType/LinkedTo/Direction）
7. **End Object**
